About The Project

The Interactive Math Visualizer is a personal passion project aiming to bridge the gap between abstract mathematical theories and intuitive visual understanding.

Built as a single-page application without heavy UI frameworks, it demonstrates proficiency in DOM manipulation, HTML5 Canvas rendering, and algorithmic implementation.
